Installation Instructions

  1. Download the DMG file using the button above
  2. Open the DMG file
  3. Drag LeetLogger to your Applications folder
  4. When first launching, right-click and select "Open" (this bypasses macOS security warnings)
  5. LeetLogger will automatically start at 9:00 AM each day with the daily problem

LeetLogger is open source. View the GitHub repository.
